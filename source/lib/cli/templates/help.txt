   Usage:

      stimpak [--flags] [...generatorNames] [--answerName="answerValue"]

   Description:

      Generate and re-generate code of any language
      using modular generators from npm!

      To search for available generators, search npm
      for keyword "stimpak" via CLI or the Npmjs.com
      website.

   Important Note:

      Flags must be set before generatorNames or
      they will be interpreted as answer values.

   Flags:
     -h, --help           output usage information
     -V, --version        output the version number
